# Requirements & Progress

## Requirements Overview
Platform for vehicle intermediation management integrated with WhatsApp. Admin panel for managing offers, suppliers, buyers, categories, distribution, and negotiation tracking.

## User Stories
- Admin can manage suppliers (CRUD with WhatsApp number)
- Admin can manage buyers with multiple categories
- Admin can manage categories for buyer groups
- Admin can create/edit/approve offers with images
- Admin can distribute offers to selected buyer categories with deduplication
- Admin can track negotiations with status workflow
- Admin can finalize offers and view history
